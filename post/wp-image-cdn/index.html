#骚操作#让你的图片用上WordPress的全球CDN

2020-04-17 · 1 min read
骚操作 Wordpress cdn 维护 日记

你是否还在为图片压缩, 服务器小水管等问题苦恼?
使用来自 Jetpack 的图片镜像CDN为你的图片加速/压缩

来自 Wordpress 的 Jetpack 提供了这样一个图片镜像CDN

https://i0.wp.com
https://i1.wp.com
https://i2.wp.com

你可以将图片Url加上 i1.wp.com 的前缀

例如

![Help Me](https://moespic.moescat.xyz/2020/04/17/h3rhuo6cezkv6woe.png)

替换为

![Help Me](https://i2.wp.com/moespic.moescat.xyz/2020/04/17/h3rhuo6cezkv6woe.png)

注意 : HTTP 链接在国内无法使用, 请使用加密 HTTPS 链接

同时这个镜像CDN提供了图片裁剪

在URL的结尾加上 resize=200%2C200 参数即可

例子

![Help Me](https://i2.wp.com/moespic.moescat.xyz/2020/04/17/h3rhuo6cezkv6woe.png?resize=200%2C200)
