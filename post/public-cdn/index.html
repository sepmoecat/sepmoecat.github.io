最近想给网站的静态资源加上CDN
但是无奈发现钱包有点空, 所以准备去寻找那些公开CDN库

本文说明

本文所用的测试工具为 17ce.com

首先是大名鼎鼎的 CDNJS

CDNJS 包含了几乎所有开源 JS 库
官网链接: https://www.cdnjs.com
CDN链接: https://cdnjs.cloudflare.com

不幸的是, CDNJS 使用的 Cloudflare 对国内用户十分不友好
延迟几乎都在200 ms 左右, 并且有很多地区屏蔽了Cloudflare的服务器
易造成无法访问.
但是! ! ! !
SB.SB博客提供了CDNJS的镜像站

https://cdnjs.loli.net/ajax/libs/

你只要把

<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>

替换为

<script src="https://cdnjs.loli.net/ajax/libs/jquery/3.2.1/jquery.min.js"></script>

JsDelivr


官网链接: https://www.jsdelivr.com/
CDN 链接: https://cdn.jsdelivr.net/
这货太牛批了, 他应该是目前最强的公共 CDN 库
他有超全的JS库以及各种 CSS 库
并且有很多骚操作, 例如用它来访问你 Github 仓库的内容
官网做了介绍. 这里是一个实例

<script src="https://cdn.jsdelivr.net/gh/sepmoecat/love.js/love.min.js"></script>

这条链接表达的是访问 Github 用户 sepmoecat 的 love.js 库的 love.min.js
也就是说, 你成功白嫖了一个全球 CDN (但是文件大小不能超过20 M )
并且这货有 ICP 备案 ! !
在国内使用网宿的CDN 速度是十分的快
