CatPic (喵图)上线啦

https://catpic.moescat.xyz

我们使用阿里云OSS存储图床图片
并使用阿里云 CDN +腾讯云 CDN +Cloudflare CDN 来加速图片的访问

但是我们没有 ICP 备案, 只能使用各大服务商的海外节点

但这也不是固定的, 我们近期发现了一些问题
腾讯云 CDN 的海外节点十分辣鸡
香港的 CDN 节点绕路到 加拿大 ??
我们在积极联系腾讯云来调度 CDN
我们海外的 CDN 使用的是 Cloudflare (不解释, 稳！）

CatPic的一些规定

  1. 你不能上传涉及色情的图片
  2. 你不可以滥用图床资源(例如上传大量无用图片）
  3. 你上传的图片不可以违反中华人民共和国与美利坚合众国falb法律的图片

如果您违反了以上规则我们会直接删除图片, 并在必需的情况下将您的信息提交给政府

我们提供了 API

您需要使用注册的账户进入我们的 API页面
https://catpic.moescat.xyz/index/api.html
API 页面提供了必要的参数以及 Token

你以使用 Picgo 的自定义插件功能连接到我们的图床系统
这里是插件的 Github: https://github.com/acewfdy/PicGoPlugins
