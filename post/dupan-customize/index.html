如何自定义百度网盘提取码

· 2020-03-21 · # 百度网盘 # 度盘 # 自定义提取码 # 百度网盘自定义密码

本站百度网盘提取码无特殊标注为 cute

你需要一个 Google Chrome 浏览器, 或者任何为 Chromium 内核的浏览器

首先你需要打开这个页面下载 Tampermonkey (油猴) 插件

如果你无法打开这个页面,你可以访问这个页面进行下载

如果你不是 Chrome 浏览器, 下载后直接将 crx 文档拖入你的浏览器里

如果你是 Chrome 浏览器,需开启开发者模式后导入 crx 文件


片刻后会有一个弹窗，点击“添加扩展程序”即可。

之后你需要打开这个页面进行安装网盘助手

最后打开你的网页版百度网盘, 点击分享文件

即可自定义网盘分享提取码
